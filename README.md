# GetNextLine

Get Next Line is a project at 42 school.

Usage:\n
  Get next line is a function that reads a file line by line.\n
  It is a function that reads a file and allows you to read a line\n
  ending with a newline character from a file descriptor. When you call\n
  the function again on the same file, it grabs the next line.\n
  This project deals with memory allocation and when to free\n
  and allocate memory to prevent leaks.\n

Return value:\n
  return 1 when it read a line\n
  return 0 when it finished reading a file\n
  return -1 when an error occurs\n
